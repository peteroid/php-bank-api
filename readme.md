## API demo

### Features
- simple bank services
- no user auth
- based on [Slim](www.slimframework.com) framework

### Tested Environment
- macOS Sierra 10.12.3
- php 7.0.17
- composer 1.4.1

### Run on your own
1. clone this repo
2. `cd php-bank-api`
3. `composer install`
4. `php -S localhost:8888 -t public public/index.php`

### Services
- Open account
- Close account
- Get current balance
- Withdraw money
- Deposit money
